# File Allocation Table (FAT)

## English
Developed as part of the **Operating Systems** university course. This project is a simulation of a FAT (File Allocation Table) file system, focusing on core operating system concepts.

### Key Features:
 ∙ Simulation of file allocation and directory structures.
 ∙ Implementation of memory management logic.
 ∙ Data serialization and efficient table lookups.

 ### Technologies:
 ∙ **Language:** C#
 ∙ **Concepts:** Operating Systems. Data Structures, File Management.
